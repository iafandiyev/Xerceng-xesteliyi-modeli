# 🩺 Xərçəng AI Analiz Sistemi

Bu layihə Streamlit və Süni İntellekt istifadə edərək  
xərçəng riskini analiz edən modern web tətbiqidir.

Sistem Breast Cancer dataset istifadə edir və  
SVM (Support Vector Machine) modeli ilə analiz aparır.
